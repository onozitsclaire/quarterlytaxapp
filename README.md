# TaxBuddy - Quarterly Tax Calculator

A friendly, dual-mode quarterly tax calculator for freelancers and gig workers.

## Features
- Simple mode for beginners
- Pro mode for experienced filers
- 2025 IRS tax rules
- Real-time calculations
- Mobile responsive
